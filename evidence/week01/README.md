# FastAPI Mini-Service
This project is a small FastAPI mini-service created for Week 01.

## Features
- Small FastAPI routers using `APIRouter`
- Input validation with Pydantic
- Endpoints:
  - `GET /health` – health check
  - `POST /echo` – echoes validated input
- Structured logging
- Automated tests with pytest

## Run the app
```bash
cd ai_prototypes/fast_api_validation
python main.py
